# The Writing Archive

A cozy AP Lang class writing archive built with React, Vite, and Firebase.

## What it does
- Lets students submit writing
- Lets students choose a name or stay anonymous
- Lets classmates leave kind notes
- Saves everything in Firebase Firestore
- Includes a simple creator mode for deleting pieces
